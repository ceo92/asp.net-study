## ASP.NET이란?
-  웹 애플리케이션을 제작할 때 사용하는 기술
- HTML CSS JS Vue JS React는 클라이언트 개발 언어, Java ,Spring, ASP.NET은 서버 개발 언어
- ASP.NET은 C#을 통해 개발
- ASP.NET은 C#의 프레임워크임 애플리케이션을 더 빠르게 개발 및 유지보수 가능
